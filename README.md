# Neon Breaker

A fast-paced, classic Brick Breaker game being built from scratch using Modern C++ and SFML.

## Description
Neon Breaker is a modern take on the classic arcade genre. The goal is to destroy all bricks on the screen using a ball and a paddle, while preventing the ball from falling off the bottom edge. 

This project is a deep dive into implementing clean game architecture, custom 2D physics, and smooth character kinematics.
